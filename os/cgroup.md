# The basics
* Provide ability to _control_ and _monitor_ resources for a process or a group
  of processes
* CG's are hierarchial, starting from a set of control subgroups
* Linux provides support for 12 controle subgroups
    - cpuset
    - cpu
    - cpuact
    - io
    - memory
    - devices
    - freezer
    - net_cls
    - net_prio
    and so on
